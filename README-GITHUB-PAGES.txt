Cara publish GitHub Pages:
1. Extract ZIP ini dahulu.
2. Upload semua fail ke root repository GitHub, bukan upload ZIP.
3. Pastikan fail index.html, app.js, styles.css dan .nojekyll berada di root.
4. GitHub > Settings > Pages.
5. Pilih Deploy from a branch.
6. Branch: main, Folder: /root.
7. Klik Save.

Nota:
- Tiada GitHub Actions workflow dimasukkan.
- Gunakan Deploy from a branch untuk elakkan ralat Jekyll action.
- Jika paparan masih data lama, tekan Ctrl + Shift + R selepas publish.
- Versi ini tanpa lencana dan tanpa nama sekolah pada paparan sistem.
- Untuk Google Sites, gunakan fail HTML/TXT embed yang disediakan.
